#### 入门
##### Ex1：读入未知数目的输入
关键语句：cin>>num; //遇到文件结束符或者输入不是字符的值（Ctrl+Z），则返回Flase
```C++
#include <iostream>
using namespace std;
int main(){
    int sum=0,value;
    while(cin>>value)
        sum=sum+value;
    cout<<"The Sum is "<<sum<<endl;
    return 0
}
```
